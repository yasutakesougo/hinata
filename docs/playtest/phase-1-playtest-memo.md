## 🔍 3大観察ポイントと記録テンプレート

### 1. 子どもが自分で選べるか（自己決定の検証）
> **検証対象**: 今日の3択（さんぽする、さんすうを1問やる、ひらがなであそぶ）
- **観察項目**:
  - [ ] 起動時に「今日の3択」が表示された際、大人の指示なしで自らどれかをタップしたか。
  - [ ] 選択するまでに迷いや躊躇があったか。
  - [ ] 自分で選んだことによって、その後のプレイへ前向きに取り組めている様子があったか。
- **具体的な様子・子どもの発話・行動の記録**:
  ```
  （例：画面が開いてすぐに「ひらがなにする！」と嬉しそうにタップした、など）
  
  ```

---

### 2. 説明なしで伝わるか（直感操作と具体表記の検証）
> **検証対象**: 10マスおやつ箱、どうぶつおやつわけ、10のともだちたいこ、さくらんぼたいじ、ひらがな盤 MVP
- **観察項目**:
  - **10マスおやつ箱**: 5のまとまりや「あといくつか」を、説明なしで視覚的に捉えて操作できたか。
  - **どうぶつおやつわけ**: どの分け方でも大正解になる仕組みに対して、楽しんで複数の分け方を試したか。
  - **10のともだちたいこ**: 叩きすぎてもペナルティがなく自動リセットされるルールに安心し、補数を体感できたか。
  - **さくらんぼたいじ**: 繰り上がり（おやつの移動アニメーションやサクランボ分解）を量感として理解している素振りがあったか。
  - **ひらがな盤 MVP**: 音声ガイドのスピードやトーンに対して、楽しそうにリピートしたり反応したか。
- **具体的な様子・つまづき箇所の記録**:
  ```
  （例：たいこをわざと連打して15回くらい叩いた後、自動で優しくリセットされて『あ、また0になった』と楽しそうにやり直していた、など）
  
  ```
