## 🔍 3大観察ポイントと記録テンプレート

### 1. 子どもが自分で選べるか（自己決定の検証）
> **検証対象**: 今日の3択（さんぽする、さんすうを1問やる、ひらがなであそぶ）
- **観察項目**:
  - [ ] 起動時に「今日の3択」が表示された際、大人の指示なしで自らどれかをタップしたか。
  - [ ] 選択するまでに迷いや躊躇があったか。
  - [ ] 自分で選んだことによって、その後のプレイへ前向きに取り組めている様子があったか。
- **具体的な様子・子どもの発話・行動の記録**:
  ```
  （例：画面が開いてすぐに「ひらがなにする！」と嬉しそうにタップした、など）
  
  ```

---

### 2. 説明なしで伝わるか（直感操作と具体表記の検証）
> **検証対象**: 10マスおやつ箱、どうぶつおやつわけ、10のともだちたいこ、さくらんぼたいじ、ひらがな盤 MVP
- **観察項目**:
  - **10マスおやつ箱**: 5のまとまりや「あといくつか」を、説明なしで視覚的に捉えて操作できたか。
  - **どうぶつおやつわけ**: どの分け方でも大正解になる仕組みに対して、楽しんで複数の分け方を試したか。
  - **10のともだちたいこ**: 叩きすぎてもペナルティがなく自動リセットされるルールに安心し、補数を体感できたか。
  - **さくらんぼたいじ**: 繰り上がり（おやつの移動アニメーションやサクランボ分解）を量感として理解している素振りがあったか。
  - **ひらがな盤 MVP**: 音声ガイドのスピードやトーンに対して、楽しそうにリピートしたり反応したか。
- **具体的な様子・つまづき箇所の記録**:
  ```
  （例：たいこをわざと連打して15回くらい叩いた後、自動で優しくリセットされて『あ、また0になった』と楽しそうにやり直していた、など）
  
  ```

### 3. 保護者が「評価」ではなく「理解」に使えるか（プロセス承認の検証）
> **検証対象**: がんばりレポート（保護者向け画面）
- **観察項目**:
  - [ ] 保護者はレポートを開いた際、正答数やスピードの「評価（○点など）」を求めていたか。
  - [ ] 「自分でこれを選べた」「ヒントを使って自分で解決できた」などのプロセス情報を見て、保護者の子どもへの声かけに温かい変化があったか。
- **保護者の発話・フィードバックの記録**:
  ```
  （例：『点数だけじゃなくて、ヒントを使って頑張ったんだねーと声をかけられて良かった』などの親の感想）
  
  ```

### 4. 【特化観察】おやつばこ テンフレーム (Phase 6A-1)
> **観察理念**: 「テスト」にしないために、子どもに質問しすぎないこと。大人は以下の声かけに留め、自然な視線や指差しの動き、発話を観察します。
> 
> * **おすすめの声かけ（最小限に）**:
>   - 「みてみようか」
>   - 「おやつ、はいったね」
>   - 「5のところで とまったね」
>   - 「もういっかい見る？」
> * **避けるべき声かけ**:
>   - 「これ何個？」
>   - 「5といくつ？」
>   - 「正解は？」
>   - 「ちゃんと見て」
> 
> - **観察指標**:
>   - 5個目のマイルストーン（一時停止）で、画面を見る時間が少し止まるか。
>   - 6〜9個目で、上段から下段へスムーズに目が移るか。
>   - 5の上段と、下段の個数を別々の塊（構造）として見ている様子があるか（1個ずつ順に数え直すか、塊で捉えているか）。
>   - 完了後に「もういっかい！」と言うか。
>   - 音量・効果音でビクッとしたり驚いたりしていないか。
>   - アニメーションが速すぎて見失っていないか。
>   - `うごきを とめる (reducedMotion)` ON の方が落ち着いて見られているか。
> 
> - **簡易記録用テンプレート**:
>   ```text
>   日付:
>   端末:
>   音: ON / OFF
>   うごきをとめる: ON / OFF
>   見た数: 5 / 7 / 10
> 
>   子どもの自然な反応（視線・発話・数え直しなど）:
>   大人の声かけ:
>   気になったこと:
>   次に変えるなら:
>   ```
