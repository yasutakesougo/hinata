# 教育監修チェックリスト（アドバイザー用）

このチェックリストは、幼児〜小学校低学年向け教育アプリ『さんすうクエスト』および『森の広場』の品質・適合性を教育監修者の皆様に評価いただくためのシートです。

チェックボックス `[ ]` に該当有無をマークし、**コメント欄**に具体的な改善案や気づき、**優先度**（必須 / できれば / 将来対応）をご記入ください。

---

## 1. 年齢適合性 (Age Appropriateness)
対象：5〜7歳のお子様にとって、画面設計や言葉づかいが直感的で適切であるか。

    *   *コメント*:
    *   *優先度*: 必須 / できれば / 将来対応

*   **[ ] 1.2 音声読み上げとテキストが一致し、聞き取りやすいスピードである**
    *   *確認点*: テキストの漢字・かな混じりと音声の読み上げが一致しているか。ロボットのような不自然な日本語発音や、早口になっていないか。

    *   *優先度*: 必須 / できれば / 将来対応
    *   *判定*: [ 合格 / 要改善 ]

    *   *確認点*: ボタンの配置が大きくて分かりやすいか。装飾が多すぎて、どこをタップすれば良いか迷う構造になっていないか。

## 4. 低刺激・安心設計 (Sensory-Friendly Design & UX)
対象：視覚・聴覚的な刺激が過度でなく、落ち着いてプレイできる工夫があるか。

    *   *確認点*: 蛍光色や原色だらけのギラギラした画面になっていないか。背景と文字のコントラストが十分で、視認しやすくなっているか。

*   **[ ] 4.2 アニメーションや効果音が過度な興奮を誘発しない**

    *   *コメント*:
