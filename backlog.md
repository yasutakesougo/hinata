### Phase 7A: 数の合成・分解と10の補数 (Number Composition & Complements)
繰り上がり・繰り下がりの学習に進む前の、数の構造理解と補数の身体化を目的としたフェーズ。
- [x] **7A-1：10マスおやつ箱 (見てわかる数の土台)**
  - [x] 上段5・下段5の10マス箱を使い、7〜10を「5とあといくつ」で視覚的に捉える機能。
- [x] **7A-2：どうぶつおやつわけ (探索的な数の分解)**
  - [x] ミミちゃん 🐰、ココちゃん 🐿 におやつを分配する。
  - [x] 特定の正解を求めず、合計が一致すれば任意の分配比率（例: 9は4と5、3と6など）をすべて肯定・大正解とする。
  - [x] 正解後に他の分け方一覧を提示し、「いろいろな分け方がある」と自然に気づかせる。
- [x] **7A-3：10のともだちたいこ (補数の身体化)**
  - [x] 太鼓のタップ回数（Web Audio API の優しいポン音）で、10にするために足りない補数を身体的に体感。
  - [x] 叩きすぎても責めずに優しく自動リセットし、探索意欲を阻害しない設計。
  - [x] どうぶつからの「8の相棒は？」に対するコール＆レスポンスと連動。
- [x] **7A-4：10をつくってから足す (繰り上がり準備のサクランボ計算)**
  - **仮テーマ**: `8 + 7` を「8と2で10、あと5」としてビジュアルで理解する。
  - **設計メモ**:
    - **画面**: `8 + 7` の数式と、10マスおやつ箱が2箱（左に8個、右に7個おやつが入っている状態）。
    - **動き**: 「10にしよう！」という声かけとともに、右の箱から2個のおやつが左の箱へ移動し、左が満杯（10個）になる。
    - **サクランボ分解の視覚化**: 右の数「7」の下に、移動した「2」と残った「5」がサクランボの枝のように線で結ばれて出現。
    - **結果の定着**: `10 と 5 で 15！` というビジュアルと数式が連動し、音声で「8はあと2で10。7から2をもらって10にしよう。10と残りの5で、15だね！」と読み上げる。
- [ ] **7A-4.1：さくらんぼたいじ実機プレイテストと微調整**
  - 実機プレイテストを実施し、子どもの反応（操作テンポ、サクランボ分解の視覚的理解度）を見て微調整する。
- [ ] **教育設計マイルストーン：設計原則の確認とメモ固定**
  - Phase 7B 移行期において、[教育設計メモ](file:///Users/yasutakesougo/Desktop/%E3%81%B2%E3%81%AA%E3%81%9F%E3%82%A2%E3%83%95%E3%82%9A%E3%83%AA/docs/education/education-design-memo.md)（数と量の一致、操作を通した身体的経験の重視）を確認・更新する。
